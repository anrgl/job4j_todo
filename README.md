# job4j_todo
## О проекте
Клиент-серверное приложение (Servlet JSP). Список дел (TODO).

- Регистрация пользователя.
- Авторизация.
- Добавление задач.

 - Общая схема клиент-серверного приложения.

 - Servlet.
 - JSP, MVC.
 - Filter, Session, Auth.
 - Клиентские языки HTML, JS, jQuery, AJAX, CSS, Bootstrap 4.
